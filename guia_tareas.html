📝 Guía del Sistema de Tareas

Organiza tu sesión de estudio con una misión activa, tareas pendientes, tareas hechas y pequeñas recompensas por avanzar ✨

🌟 ¿Qué es el sistema de tareas?

El sistema de tareas permite que cada persona del chat apunte qué está haciendo durante la sesión. Sirve para estudiar con intención, tener una misión clara y celebrar los avances.

Cada usuario puede tener una tarea activa, varias tareas pendientes y un historial de tareas hechas.

🎯 La tarea activa

La tarea activa es tu misión principal ahora mismo. Es lo que el bot entiende que estás intentando completar.

!tarea hacer resumen del tema 4

Si no tenías tarea activa, se crea una nueva. Si ya tenías una tarea activa, la anterior pasa automáticamente a pendientes para que no se pierda.

Ejemplo: si tu tarea activa era “leer apuntes” y escribes !tarea hacer esquema, “leer apuntes” pasa a pendientes.

✅ Completar una tarea

Cuando termines tu tarea activa, usa:

!hecha

El bot marca la tarea como hecha, guarda la fecha y te da puntos como recompensa.

Ahora mismo completar una tarea da +15 puntos 💎.

Si tenías tareas pendientes, el sistema puede subir la primera pendiente como nueva tarea activa. Así siempre tienes una siguiente misión preparada.

📋 Ver tus tareas

Para ver tu lista completa:

!mistareas

El bot muestra tus tareas ordenadas así:

[ACTIVA] — la tarea que estás haciendo ahora.
[PENDIENTE] — tareas guardadas para después.
[HECHA] — tareas completadas.

🎯 Ver tu tarea actual

Si te despistas o no recuerdas qué estabas haciendo, usa:

!tareaactual

El bot te recordará tu misión activa.

🔁 Cambiar de tarea

Puedes cambiar tu tarea activa por otra tarea de tu lista usando su número. Primero mira tu lista:

!mistareas

Después elige el número:

!cambiar 2

La tarea elegida pasa a ser activa. La tarea activa anterior vuelve a pendientes.

✏️ Editar una tarea

Si escribiste mal una tarea o quieres aclararla, puedes editarla:

!editar 2 leer tema 5 y hacer esquema

El número indica qué tarea quieres editar, y el texto de después será el nuevo contenido.

Si ya tienes una tarea activa o pendiente con ese mismo texto, el bot evita duplicarla.

🔄 Retomar una tarea

Si quieres volver a una tarea guardada, usa:

!retomar 3

Esa tarea pasa a ser tu tarea activa. Si había otra activa, vuelve a pendientes.

🧹 Limpiar tareas hechas

Para quitar de tu lista las tareas que ya están hechas:

!limpiar

Esto solo limpia tareas completadas. Las tareas activas y pendientes se conservan.

📅 Ver tareas hechas hoy

Para consultar lo que ya has terminado durante el día:

!hechashoy

El bot mostrará las tareas que hayas completado hoy.

📌 Estados de las tareas

activa

Tu misión actual. Normalmente solo hay una por usuario.

pendiente

Una tarea guardada para hacer más tarde.

hecha

Una tarea completada. Se guarda con fecha y puede dar puntos.

📋 Comandos principales

!tarea texto — añade una nueva tarea activa.

!hecha — completa la tarea activa y gana puntos.

!mistareas — muestra tu lista completa.

!tareaactual — muestra tu misión activa.

!cambiar número — cambia tu activa por otra tarea.

!editar número texto — modifica una tarea.

!retomar número — retoma una tarea de tu lista.

!limpiar — limpia tareas hechas.

!hechashoy — muestra las tareas completadas hoy.

!infotareas — muestra resumen y enlace a esta guía.

💚 Filosofía del sistema

Este sistema no está pensado para presionarte, sino para ayudarte a avanzar con calma. Una tarea pequeña y clara puede cambiar una sesión entera.

Puedes cambiar de tarea, dejar cosas pendientes, retomar más tarde y celebrar cada avance. Lo importante no es hacerlo perfecto: es seguir volviendo 🌿
